> 本文档提供基于工具实际功能的教学场景设计，每个示例都对应软件的真实操作和预期观察结果。

## 示例一：探索径向节点——3s轨道的三个峰

**适用模式**：单轨模式（Single）  
**适用课程**：普通化学 / 结构化学  
**核心概念**：径向波函数的节点、最可几半径、径向分布函数

### 操作步骤

1. 将模式切换到「单轨」
2. 在轨道列表中选择「3s」
3. 点击「启动」，观察点云逐渐生成
4. 打开右侧「数据」面板，观察径向分布函数（RDF）图表

### 预期观察

- **三维点云**：呈现球对称分布，但密度分布不均匀
- **RDF曲线**：随采样点数增加，直方图逐渐收敛于理论曲线，呈现**三个明显的峰**
- **最外峰最高**：这是最可几半径——电子最可能出现的位置

### 引导问题

> "为什么3s轨道有三个峰？这与量子数n有什么关系？"

**关键洞察**：径向节点数 = n - l - 1 = 3 - 0 - 1 = 2。两个节点将径向分布分成三个区域（三个峰）。

## 示例二：发现子壳层球对称性——2p轨道的"消失"

**适用模式**：多选模式（Multi）  
**核心概念**：完整子壳层的球对称性、角向分布的正交性

### 操作步骤

1. 切换到「多选」模式
2. 依次选择「2px」「2py」「2pz」三个轨道
3. 点击「启动」

### 预期观察

- **三维点云**：原本预期的三个"哑铃"消失了！取而代之的是**球形分布**
- **RDF曲线**：仍然是2p轨道的特征曲线（单峰），说明径向部分没有变化

### 引导问题

> "为什么三个p轨道叠加后变成了球？教科书说的'完整子壳层球对称'是这个意思吗？"

**关键洞察**：这是因为 |Y₁₁|² + |Y₁₋₁|² + |Y₁₀|² = 常数。对于完整的l子壳层，角向分布之和是各向同性的。

**教学延伸**：让学生尝试只选择两个p轨道（如2px + 2py），观察结果不再是球形，从而理解"完整"子壳层的含义。

## 示例三：感受有效核电荷——跨原子比较2p轨道

**适用模式**：比照模式（Compare）  
**核心概念**：有效核电荷、屏蔽效应、轨道收缩

### 操作步骤

1. 切换到「比照」模式
2. 在第一行选择「2p」轨道，原子选择「H」（这是氢原子的激发态）
3. 在第二行选择「2p」轨道，原子选择「C」（碳的价层2p）
4. 在第三行选择「2p」轨道，原子选择「Si」（硅的内层2p）
5. 点击「启动」

### 预期观察

- **三种颜色的点云**（红、绿、蓝）**重叠**在同一个三维空间的原点
- **尺寸差异通过颜色分层直观呈现**：
  - H-2p（红）：最弥散，覆盖范围最大
  - C-2p（绿）：中等大小
  - Si-2p（蓝）：最紧凑，集中在原点附近
- **RDF图表**：三条理论曲线的峰位置依次向原点收缩

### 引导问题

> "同样是2p轨道，为什么在不同原子中大小差别这么大？"

**关键洞察**：有效核电荷 Zeff 随原子序数增加而增加，导致电子被更强地束缚，轨道收缩。硅的2p是内层轨道，受到的屏蔽更少，Zeff 更大。

## 示例四：理解sp³杂化——从原子轨道到四面体

**适用模式**：杂化模式（Hybrid）  
**核心概念**：轨道杂化、四面体几何、等价杂化轨道

### 操作步骤

1. 切换到「杂化」模式
2. 选择「2s」「2px」「2py」「2pz」四个轨道
3. 点击「启动」，观察点云生成
4. 等待渲染完成后，右侧「杂化」面板会出现4个按钮：「杂化轨道1」「杂化轨道2」「杂化轨道3」「杂化轨道4」
5. 分别点击按钮，切换显示不同的杂化轨道

### 预期观察

- **四个杂化轨道分别指向四面体的四个顶点**
- **每个杂化轨道都是不对称的"胖水滴"形状**（大瓣+小瓣）
- **点击不同按钮，可以看到四个方向依次亮起**
- **勾选「轨道轮廓(95%)」可以看到清晰的等值面边界**

### 引导问题

> "教科书告诉我们sp³杂化是四面体构型。但为什么是四面体？能不能是其他形状？"

**关键洞察**：四面体构型不是人为规定的，而是使4个等价杂化轨道相互排斥最小化的结果。软件计算使用的系数矩阵（每个系数±0.5）正是正交条件的数学解。

**教学延伸**：让学生只选择3个轨道（如2s + 2px + 2py），观察sp²杂化的平面三角形指向。

## 示例五：探索非正交杂化——1s + 2s 的干涉效应

**适用模式**：杂化模式（Hybrid）  
**核心概念**：交叉项（干涉项）、非正交轨道叠加

### 操作步骤

1. 切换到「杂化」模式
2. 选择「1s」和「2s」两个轨道
3. 点击「启动」

### 预期观察

- **RDF曲线呈现明显的干涉条纹**——这与分别显示1s和2s的简单叠加完全不同
- **三维点云出现"明暗相间"的壳层结构**

### 物理原理

这个组合在化学上没有意义（1s和2s不参与同一杂化），但物理上完全合法。1s和2s具有相同的角向部分（都是s轨道，l=0, m=0），因此波函数叠加时产生交叉项：

$$P(r) = r^2 \times |c_1 R_{1s} + c_2 R_{2s}|^2 = r^2 \times (c_1^2 R_{1s}^2 + c_2^2 R_{2s}^2 + 2c_1 c_2 R_{1s} R_{2s})$$

最后一项就是交叉项（干涉项）。它可正可负，导致某些区域概率增强、某些区域概率减弱。

### 引导问题

> "为什么化学里不用1s+2s这种杂化？"

**关键洞察**：能量差距太大（1s是核心层，2s是价层），没有化学意义。但物理引擎仍然允许你做这个"实验"，观察干涉效应——这正是「兜底原则」的体现。

## 示例六：波函数的相位——看见"正"和"负"

**适用模式**：单轨模式（Single）  
**核心概念**：波函数的符号、节面、Condon-Shortley相位约定

### 操作步骤

1. 选择「2pz」轨道
2. 勾选「相位显示」开关
3. 点击「启动」

### 预期观察

- **点云分为两种颜色**：一半是蓝色（ψ > 0），一半是红色（ψ < 0）
- **两个"瓣"分别位于z轴的正负两侧**
- **xy平面上没有点**——这是**节面**（ψ = 0）

### 引导问题

> "为什么p轨道有正负两瓣？电子的概率密度不是|ψ|²吗，怎么会有负的？"

**关键洞察**：概率密度永远是正的（|ψ|² ≥ 0），但波函数本身可以有正负。相位信息在成键时至关重要——同相重叠形成成键轨道，反相重叠形成反键轨道。

**教学延伸**：选择「3dz²」轨道开启相位显示，观察其独特的"两极+环"结构（中心环的相位与两极相反）。

## 示例七：大数定律的活教材——观察收敛过程

**适用课程**：统计力学概论 / 量子化学实验  
**核心概念**：大数定律、统计涨落、概率→频率

### 操作步骤

1. 选择任意轨道（如「2p」）
2. 将「最大点数」设置为「200,000」或更高
3. 将「生成速度」设置为中等
4. 点击「启动」，**持续观察RDF图表的变化**

### 预期观察

| 采样阶段 | RDF图表表现 |
|---------|------------|
| 100点附近 | 直方图极度不规则，完全偏离理论曲线 |
| 1,000点附近 | 大致形状开始出现，但波动明显 |
| 10,000点附近 | 与理论曲线较好吻合，峰位正确 |
| 100,000点以上 | 几乎完全重合 |

### 引导问题

> "为什么采样点数越多，结果越接近理论值？这与物理测量有什么关系？"

**关键洞察**：这正是量子力学的核心：**单次测量是随机的，但多次测量的统计规律是确定的**。概率密度函数 |ψ|² 告诉我们的是"如果做无穷多次测量，各位置出现的频率"。
