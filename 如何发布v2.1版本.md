## 🚀 发布步骤（详细图文教程）

### 第一步：访问发布页面

1. 打开浏览器，访问项目仓库：
   ```
   https://github.com/benbaobaoshigemi/Hydrogen-Atom-Electron-Cloud-Visualization-REV
   ```

2. 点击页面右侧的 **"Releases"** 链接
   - 或直接访问：https://github.com/benbaobaoshigemi/Hydrogen-Atom-Electron-Cloud-Visualization-REV/releases

3. 点击 **"Draft a new release"** 或 **"Create a new release"** 按钮

---

### 第二步：填写发布信息

#### 1. 选择标签（Tag）
- 在 **"Choose a tag"** 下拉框中，输入：`v2.1`
- 如果标签不存在，会显示 **"Create new tag: v2.1 on publish"**
- 确保目标分支是 `main`（或 `master`）

#### 2. 填写发布标题
在 **"Release title"** 输入框中填写：
```
氢原子电子云可视化 v2.1 - UI 视觉全面升级
```

#### 3. 填写发布说明
在 **"Describe this release"** 大文本框中：
- **方法一（推荐）**：复制 `RELEASE_DESCRIPTION_v2.1.txt` 文件的完整内容
- **方法二**：手动输入或编辑发布说明

**快速复制命令**（如果您在项目目录下）：
```bash
cat RELEASE_DESCRIPTION_v2.1.txt
```
然后复制输出的全部内容。

---

### 第三步：上传可执行文件

1. 在发布页面底部找到 **"Attach binaries by dropping them here or selecting them."** 区域

2. **方法一：拖放上传**
   - 找到项目目录中的 `Hydrogen_Electron_Cloud.exe` 文件
   - 拖动文件到上传区域

3. **方法二：点击选择**
   - 点击 **"selecting them"** 链接
   - 在文件浏览器中选择 `Hydrogen_Electron_Cloud.exe`

4. 等待文件上传完成（文件大小约 9.4MB，可能需要几秒到几分钟）

---

### 第四步：设置发布选项

在页面底部，确认以下选项：
- ✅ **不要**勾选 "Set as a pre-release"（这不是预发布版本）
- ✅ **要**勾选 "Set as the latest release"（这是最新版本）
- ⭕ "Create a discussion for this release"（可选，建议勾选以便用户讨论）

---

### 第五步：发布！

1. 检查所有信息填写正确：
   - 标签：v2.1
   - 标题：氢原子电子云可视化 v2.1 - UI 视觉全面升级
   - 说明：完整的发布说明
   - 附件：Hydrogen_Electron_Cloud.exe 已上传

2. 点击绿色的 **"Publish release"** 按钮

3. 等待几秒钟，GitHub 会处理您的发布

## 🆘 常见问题

### Q1: 上传可执行文件失败怎么办？
**A:** 
- 检查网络连接
- 确认文件大小不超过 2GB（我们的文件只有 9.4MB，没问题）
- 尝试换个浏览器（推荐 Chrome 或 Firefox）
- 如果仍然失败，可以先发布不带附件，之后编辑发布添加附件

### Q2: 发布后发现错误怎么办？
**A:** 
- GitHub Release 可以编辑
- 访问发布页面，点击右上角 "Edit release" 按钮
- 修改内容后重新保存

### Q3: 如何删除发布？
**A:** 
- 访问发布页面
- 点击 "Delete" 按钮（谨慎操作！）
- 注意：删除发布不会删除 Git 标签，需要单独删除标签

### Q4: 标签打错了怎么办？
**A:** 
- 如果还没发布，直接修改标签名称
- 如果已经发布，需要：
  1. 删除发布
  2. 删除标签（git tag -d v2.1 && git push origin :refs/tags/v2.1）
  3. 重新创建发布
